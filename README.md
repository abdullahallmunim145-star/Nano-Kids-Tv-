# 🎬 NANO KIDS - Bengali TV Player

একটি সুন্দর এবং আধুনিক বাংলা টিভি প্লেয়ার যা বিশেষভাবে শিশুদের জন্য ডিজাইন করা হয়েছে।

## ✨ ফিচার

- 📺 **ফুল স্ক্রিন ভিডিও প্লেয়ার** - সম্পূর্ণ স্ক্রিন জুড়ে ভিডিও প্লেব্যাক
- 🎨 **প্রফেশনাল ডিজাইন** - আধুনিক এবং ব্যবহারকারী-বান্ধব ইন্টারফেস
- 📜 **নিউজ টিকার** - নিচে স্ক্রল করা সংবাদ/ঘোষণা
- 🎯 **লোগো ডিসপ্লে** - ব্র্যান্ড লোগো উপরের ডানকোণে
- 📱 **রেসপন্সিভ** - সকল ডিভাইসে সঠিকভাবে কাজ করে

## 🚀 শুরু করুন

### প্রয়োজনীয়তা
- যেকোনো আধুনিক ওয়েব ব্রাউজার (Chrome, Firefox, Safari, Edge)
- ইন্টারনেট সংযোগ

### ব্যবহার

1. **ফাইল ডাউনলোড করুন**
   ```bash
   git clone https://github.com/abdullahallmunim145-star/Nano-Kids-Tv-.git
   ```

2. **ব্রাউজারে খুলুন**
   - `index.html` ফাইলটি ডাবল-ক্লিক করুন অথবা
   - ব্রাউজারে ড্র্যাগ করুন

## ⚙️ কাস্টমাইজেশন

### লোগো পরিবর্তন করুন
`index.html` ফাইলে এই লাইনটি খুঁজুন এবং আপডেট করুন:
```html
<img src="https://i.ibb.co/your-logo-link.png" class="logo" alt="NANO KIDS">
```
আপনার লোগো URL দিয়ে `https://i.ibb.co/your-logo-link.png` বদলে দিন।

### ভিডিও লিঙ্ক পরিবর্তন করুন
```html
https://drive.google.com/file/d/1grNQDi_RfgJKEYdJSojpIUh_C2hBfdFr/view?usp=drivesdk
```
আপনার ভিডিও URL দিয়ে বদলে দিন।

### নিউজ টিকার টেক্সট পরিবর্তন করুন
```html
<marquee behavior="scroll" direction="left" scrollamount="8">
  আপনার টেক্সট এখানে লিখুন ...
</marquee>
```

### স্টাইল পরিবর্তন
- **লোগো সাইজ**: `width: 150px;` পরিবর্তন করুন
- **নিউজ টিকার রঙ**: `border-top: 4px solid #ffcc00;` পরিবর্তন করুন
- **টেক্সট সাইজ**: `font-size: 22px;` পরিবর্তন করুন
